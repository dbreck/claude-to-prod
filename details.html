03 — End to End

Idea to live URL

The actual sequence from "I have an idea" to "here's the link."

1
Brainstorm in Claude Chat claude.ai

Describe what you want to build. Get help planning the structure, stack, features, and copy. Use artifacts to prototype ideas visually — no code on your machine yet.

2
Set up with Cowork Desktop

Use Cowork to organize project files, move assets, or run light automation on your local machine — without opening a terminal.

3
Build with Claude Code Terminal

Create a project folder, open your terminal, and run Claude Code. Give it your plan from step 1 and watch it build the files.

# install once
npm install -g @anthropic-ai/claude-code

# run it in your project
mkdir my-project && cd my-project
claude "build a portfolio site with hero, about, and contact"
4
Review & iterate Claude Code

Open the files in VS Code. Use the Claude Code extension to tweak, fix, and refine. Keep prompting until it's right — Claude remembers your project context.

5
Push to GitHub git

Initialize a repo and push your code. You can ask Claude Code to handle the git init, commit, and push — or do it yourself.

git init
git add . && git commit -m "initial build"
git remote add origin https://github.com/you/my-project.git
git push -u origin main
6
Deploy on Vercel vercel.com

Go to vercel.com, import your GitHub repo, click Deploy. You get a shareable URL within 60 seconds — and every future push auto-deploys.

04 — Jump In

Get started today

Pick your entry point based on where you're at.

Never used AI tools

  1. Go to claude.ai and sign up (free)
  2. Start a conversation: "Help me plan a portfolio website"
  3. Try creating an artifact — ask for a mockup or mini-app
  4. Download the desktop app for Cowork

Ready to build

  1. Install Node.js from nodejs.org
  2. Run npm install -g @anthropic-ai/claude-code
  3. Create a folder and run claude inside it
  4. Describe what you want — Claude builds it

Built something, want to ship

  1. Create a GitHub account (free)
  2. Ask Claude Code: create a github repo and push
  3. Go to vercel.com, import the repo
  4. Click Deploy → share the URL

Want to level up

  1. Install the Claude Code VS Code extension
  2. Add a CLAUDE.md to your project root
  3. Try multi-file tasks: "refactor this into components"
  4. Explore MCP servers for extra capabilities
